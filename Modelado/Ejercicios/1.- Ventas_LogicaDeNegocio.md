# 1, 2, 4, 5, 6 y 7:

## Clientes (CRUD) **ED**

- id_cliente: int **PK**
- nombre_cliente: varchar[50]
- apellido_cliente: varchar[50]
- teléfono: varchar[20] **UQ**
- email: varchar[50] **UQ**
- direccion_cliente: varchar[500]
- codigo_postal: int
- ciudad: varchar[50]
- id_país: int **FK**

## Productos (CRUD) **ED|EC**

- id_producto: int **PK**
- nombre_producto: varchar[50]
- descripción_producto: varchar[500]
- precio_producto: decimal[10,2]
- cantidad_disponible_producto: int

## Ventas (CRU) **ED**

- id_venta: int **PK**
- id_cliente: int **FK**
- id_estado: int **FK**

## País (CRUD) **EC**

- id_pais: int **PK**
- nombre_país: varchar[50]
- dominio_país: varchar[10] **UQ**

## Detalle_Venta (CRU) **EP**

- id_venta: int **PK|FK**
- id_producto: int **PK|FK**
- cantidad_producto: int
- precio_venta: decimal[10,2]

## ESTADO_VENTA **EC** (CRUD)

- id_estado: int **PK**
- nombre: varchar[20]

# 8 Relaciones:

1. Un **cliente** pertenece a un **país**, y un **país** tiene **clientes** (1 - M)
2. Una **cliente** hace **ventas** (1 - M)
3. Una **venta** tiene **estados** (1 - M)
4. Una **venta** tiene **detalles** (1 - M)
5. Un **detalle** tiene **productos** (1 - M)
