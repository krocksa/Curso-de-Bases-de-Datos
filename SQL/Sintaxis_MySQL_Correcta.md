# Documento 1: Tipos de Datos en MySQL

| Nombre | Significado |
| :--- | :--- |
| `TINYINT` | Entero muy pequeño de 1 byte. |
| `INT` | Entero estándar de 4 bytes. |
| `BIGINT` | Entero grande de 8 bytes para valores masivos. |
| `DECIMAL(p,s)` | Número exacto con precisión (p) y escala (s) fija. |
| `FLOAT` | Número de punto flotante de precisión simple. |
| `DOUBLE` | Número de punto flotante de precisión doble. |
| `VARCHAR(n)` | Cadena de texto de longitud variable hasta (n) caracteres. |
| `TEXT` | Cadena de texto de gran tamaño. |
| `DATE` | Valor de fecha en formato año-mes-día. |
| `DATETIME` | Combinación de fecha y hora. |
| `TIMESTAMP` | Marca de tiempo que registra cambios automáticamente. |
| `BOOLEAN` | Valor lógico de verdadero o falso (0 o 1). |
| `JSON` | Formato para almacenar objetos de datos estructurados. |

# Documento 2: Operaciones en MySQL

| Nombre | Significado |
| :--- | :--- |
| `SELECT` | Comando para extraer o leer datos de la base de datos. |
| `INSERT INTO` | Comando para añadir nuevos registros a una tabla. |
| `UPDATE` | Comando para modificar los valores de registros existentes. |
| `DELETE` | Comando para borrar registros de una tabla. |
| `CREATE TABLE` | Comando para generar una nueva tabla. |
| `ALTER TABLE` | Comando para cambiar la estructura de una tabla existente. |
| `DROP TABLE` | Comando para eliminar permanentemente una tabla. |
| `WHERE` | Cláusula para filtrar los resultados de una operación. |
| `GROUP BY` | Cláusula para clasificar filas según valores comunes. |
| `HAVING` | Cláusula de condición aplicada tras un agrupamiento. |
| `ORDER BY` | Cláusula para ordenar resultados (ASC o DESC). |
| `LIMIT` | Cláusula para restringir el número de filas devueltas. |

# Documento 3: Funciones y Métodos en MySQL

| Nombre | Significado |
| :--- | :--- |
| `COUNT()` | Función que devuelve el número total de filas. |
| `SUM()` | Función que calcula la sumatoria de valores numéricos. |
| `AVG()` | Función que calcula el valor promedio de una columna. |
| `MAX()` | Función que obtiene el valor más alto de un conjunto. |
| `MIN()` | Función que obtiene el valor más bajo de un conjunto. |
| `CONCAT()` | Función que une múltiples cadenas de texto. |
| `UPPER()` | Función que transforma texto a letras mayúsculas. |
| `LOWER()` | Función que transforma texto a letras minúsculas. |
| `LENGTH()` | Función que cuenta la cantidad de caracteres de una cadena. |
| `NOW()` | Función que devuelve la fecha y hora actual del sistema. |
| `DATEDIFF()` | Función que calcula la diferencia en días entre dos fechas. |
| `IF()` | Función que devuelve un valor u otro según una condición. |
| `COALESCE()` | Función que retorna el primer valor no nulo de una lista. |
