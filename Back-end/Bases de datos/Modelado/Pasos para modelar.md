## GLOSARIO

- **PK**: PRIMARY KEY
- **FK**: FOREING KEY
- **UQ**: UNIQUE ATRIBUTE
- **ED**: ENTIDAD DE DATOS
- **EP**: ENTIDAD PIVOTE
- **EC**: ENTIDAD CATÁLOGO
