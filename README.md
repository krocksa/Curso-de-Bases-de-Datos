# Curso Full Stack

Este repositorio contiene el material de estudio y proyectos prácticos desarrollados durante mi formación como **Desarrollador Full Stack**. Este proyecto integra tecnologías de frontend moderno con arquitecturas robustas de backend y gestión de bases de datos.

## 🚀 Tecnologías Utilizadas

### Frontend
*   **HTML5 / CSS3**: Estructura y diseño web responsivo.
*   **Bootstrap**: Framework para componentes de interfaz y diseño ágil.
*   **JavaScript**: Lógica de cliente y dinamismo.

### Backend
*   **C#**: Lenguaje principal de desarrollo.
*   **.NET / ASP.NET Core**: Framework para la construcción de servicios y aplicaciones robustas.

### Bases de Datos y ORM
*   **SQL Server**: Sistema de gestión de bases de datos relacionales.
*   **Entity Framework**: ORM para el mapeo y manejo de datos en .NET.

---

## 📚 Objetivos del Curso
El enfoque de este curso es transformar conocimientos base de datos en aplicaciones completas, permitiendo la comunicación entre la capa de presentación (frontend) y la capa de datos (backend), garantizando aplicaciones escalables y mantenibles.

## 🛠 Estado del Proyecto
Actualmente en desarrollo activo, enfocándome en la integración completa de todas las tecnologías listadas.
