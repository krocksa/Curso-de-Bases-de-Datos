# Curso de Bases de Datos

Repositorio con el contenido, ejercicios y guías prácticas del curso de bases de datos.

## Estructura del Repositorio

* **/Modelado**: Conceptos de diseño, lógica de negocio y diagramas.
* **/SQL**: Sintaxis, consultas, uniones (JOINs) y funciones en MySQL.
* **/NoSQL**: Conceptos y manejo de bases de datos no relacionales.

## Tecnologías Utilizadas
* MySQL / MySQL Workbench
* Markdown para documentación
